# Database (Dataset for Teacher)

This folder is used to store the **dataset generated by the project** while it runs.

## What is stored

- **Predictions dataset**: every prediction request creates a row.
- **Actual outcome**: after 15 minutes, the row is updated with actual price + correct/wrong.
- **Training samples**: if the prediction is wrong, the feature row + target label is stored for retraining.

## Files

- `stock_prediction.sqlite3`: SQLite database created automatically when backend runs.
- `schema.sql`: database schema (tables/columns) used by the system.

## How to show your teacher

- Open `stock_prediction.sqlite3` using any SQLite viewer.
- Show tables:
  - `predictions`
  - `training_samples`
